# Power BI Report Style Guide

#### Style guide for consistent look and experience across reports so that the reporting portal feels more cohesive and professional.

- Background
1. Background remains white (#FFFFFF) 

- Header image
2. Header should be for the page navigation and last refresh indicator

- Spacing
1. Whenever possible, use the grid inside Power BI desktop editor. Make sure to use the Format > Align to get elements to align middle and distribute vertically or horizontally as needed

- Typography
1. Unless otherwise requested, we use the default font called DIN

- Font sizing

- In-Report Navigation

- Filter control placement
1. Filters that affect the entire page like the date range or the row granularity should be underneath the header distributed horizontally

- Location of filters
1. Filters that affect the content (division, region, store, employee, etc.) should remain on the left in a vertical format

- Reset filter buttons
1. The template file will have the button ready to go that has a white default background and transparent red color on hover to indicate an irreversible change to the user
2. Microcopy: "Click the button below to reset all filters and dates."

- Chart coloring
  Plese refer to the COLORS.md file for color guidelines.

- Panel use and bordering
1. Allow 10 to 15 pixels between visuals and whitespace around the sides and the bottom
2. Make any rounded corners at least 5px
